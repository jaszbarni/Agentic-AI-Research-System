## Kivonat:
A feladatom egy kutatás segítő program elkészítése volt, a programban ArXiv könyvtár segítségével egy MI ágensen keresztül lehet lekérdezni és letölteni publikációkat. Ezeket meg lehet tekinteni és egy másik MI ágens egy összefoglalót ír, ez alapján kiemeli a fontos szavakat a dokumentumból. 

### Kulcsszavak:
Multiágens rendszer, Pydantic BaseModel, Phoenix Arize, Eszközhasználat, MCP
